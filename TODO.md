## Frontend

- Cost 페이지
  - 코스트 회복의 정확성 검증 및 수정
  - 학생 정보를 받아와서 선택하는 기능
  - 코스트 관련 기믹 구현
    - 코스트 회복량: 체리노, 히마리, 새루카, 수시노, 히후미, 슌, 바스나
    - 코스트 변경: 우이, 코코나, 새후카, 토키, 체스미
  - 시작 시 타이머 시간 기준 2초간 코스트 회복 연기
  - 타이머 추가
    - 타이머 시간 선택 기능
- Damage 페이지
  - 학생 정보를 받아오기
- Updates 페이지
  - 역대 및 향후 업데이트 내역을 주 단위로 알려주는 페이지 
- URL `#` 문제 해결
  - 서버에서 정적 파일 넘기는 방식으로 변경
